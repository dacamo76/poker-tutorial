  1. Actions
    Plain objects that propagate through the app triggering changes and passing source changes.
  2. Reducers
    Pure functions that place data from actions into the application data store, often with some manipulation.
  3. Store
    Immutable single object tree that maintains all application-level data.
  4. React
    View components that present an interface to the user, visualize Redux Store, provide controls for the user to kick off actions.
